K2-FSA sherpa-onnx ASR is an Onnx AI model deployed on Android or other devices, supporting offline speech-to-text conversion in Chinese and English.


Sherpa is an open-source speech processing framework developed by the K2-FSA team. It aims to address the complexities of model deployment and cross-platform adaptation in traditional speech recognition tools such as Kaldi. By integrating modern deep learning technologies and an efficient inference engine, it provides a one-stop solution from speech recognition and synthesis to speaker recognition, excelling particularly in real-time performance, low resource consumption, and cross-platform compatibility.
